切片是 Python 提供的一种高级语法，用来取出序列类型的指定数据，比如：字符串、列表、元组、字典

切片通过索引来指定范围，从一个大的序列类型切出小的序列类型数据

slice[开始索引:结束索引:步长]

步长用于指定隔几个索引取一次数据，可以省略不写，默认为1，当步长为正时，从左至右切片，为负时，从右至左切片

```python
str = "0123456789"
str[2:6] # 2345
str[2:] # 23456789
str[:4] # 0123
str[:] # 0123456789
str[::2] # 02468
str[1::2] # 13579
str[::-1] # 9876543210
str[-1:-6] # 98765
```

::: tip
当步长为负时，开始索引应该从 -1 开始，否则无法取出数据
:::
