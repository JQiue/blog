因为 Vue 是基于 MVVM 设计模式实现的，所以也提供了双向数据传递，Vue 中通过 v-model 指令来实现双向绑定，但只适用于`input`，`textarea`，`select`元素

```html
<input type="text" v-model="">
```

假设将上述页面绑定到 msg 中

```javascript
let app = new Vue({
  el: "#app",
  data: {
    msg: "JQiue"
  }
})
```

::: demo
<p>{{msg}}</p>
<input type="text" v-model="msg">

<script>
export default {
  data(){
    return {
      msg: "JQiue"
    }
  }
}
</script>
:::
