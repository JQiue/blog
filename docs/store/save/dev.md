## 前端自动化

**Gulp**

```shell script
npm i gulp -d
```

**Grunt**

```shell script
npm i grunt -d
```

**Webpack3**/**Webpack4**

```shell script
npm i webpack@3 -d

# webpack4中将cli分离了出来
npm i webpack-cli -d
npm i webpack@4 -d
```

**Browserify**

```shell script
npm install -g browserify
```
