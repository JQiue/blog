## 排版

本博客统一采用中文文案，排版的相关用法，降低访问者的沟通成本，并约束所有的文档

+ 空格
  + 中英文之间添加空格
  + 中文和数字之间添加空格
  + 数字和单位之间不添加空格
  + 全角标点和其他字符之间不加空格
+ 全角和半角
  + 使用全角中文标点
  + 数字使用半角字符
  + 完整的英文整句、特殊名词、在其内容中使用半角
