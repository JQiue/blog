配置文件可以用来告诉webpack来如何做任务，配置是标准的CommonJS规范，且导出一个JavaScript对象

通常将`webpack.config.js`作为配置文件名并放在项目根目录，由webpack根据对象定义的属性进行解析

```javascript
const path = require('path');
module.exports = {
  mode: 'development',
  entry: './main.js',
  output: {
    path: path.resolve(__dirname, 'dist'),
    filename: 'bundle.js'
  }
};
```
